** Seq: Sequence **

Generates a sequence of values in a Row matrix

Parameters:
    Z: Initial value
    Y: Increment value
    X: Number of iterations
    
Example:
    10 <Enter>
    2 <Enter>
    6 <Seq>
    
Output:
    X: [10 12 14 16 18 20]
    
Notes:
    - Any type that supports addition '+' is supported
